1. ✅ 상세 기능 설계
A. 홈 화면 (3분할 카드)
나는 누구?

감정 카드

상담 시작

B. “나는 누구?” 모듈
관심사 대분류 키워드 제시 (8개 카테고리)

선택 시 소주제 키워드 (예: 음악 → 인디, 클래식, 힙합 등)

이상형 월드컵 방식: 본인의 선호 확인

성향 & 관심사 탐색 기록 저장

결과 기반 → 자아 진단 요약 제공

C. 감정 카드
6대 감정 범주 + 하위 키워드 100+ 포함

오늘의 감정 기록 (😃 / 😐 / 😢 / 😡 등 이모지 기반)

주간/월간 감정 변화 시각화 제공

자동 추천: “당신은 최근 불안 키워드가 증가하고 있습니다.”

D. 상담 모듈
키워드 선택형 고민 입력 (ex. 우울, 동기부여, 연애 등)

GPT 상담 시작

페르소나 3종 자동 추천 (동적 생성)

MBTI, 감정 기록, 자아 탐색, 관심사 기반 분석

예:

🎯 냉철한 전략가

🧘 감성 공감형

🎩 유쾌한 조언가

사용자가 선택 → GPT 상담 방식 적용

E. 테스트 탭
대분류:

전문 심리검사 (예: 우울 지수, 자기효능감)

흥미용 심리테스트 (예: 연애성향, 타로 심리)

소주제:

카드형 테스트 UI 제공

완료 시 결과 요약 & 상담 연결 가능

F. 카테고리 탭
캘린더: 감정 기록 + 상담 일지 요약

피드백: 월간 변화 차트 + 감정/성향 변화 분석

운세: 사주, 타로, 월간/오늘 운세

콘텐츠: 기사/영상 큐레이션 (성향 기반 추천)

G. 관리자 페이지
관리자 로그인 (일반 로그인과 분리)

대시보드 기능:

콘텐츠 업로드

사용자 피드백 관리

유입 통계 확인 (신규/재방문)

구독 현황 (무료 vs 유료)

매출 흐름 요약 (월간/누적/그래프)

시스템 설정 (공지, 점검 등)
